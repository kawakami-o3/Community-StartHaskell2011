n+kパターンの｛変換｝｛へんかん｝
---------------------------------

この｛章｝｛しょう｝はHaskellから｛廃止｝｛はいし｝された「n+kパターン」を
｛大々的｝｛だいだいてき｝に｛使｝｛つか｝います。｛例｝｛たと｝えば、
57ページに｛定義｝｛ていぎ｝している`factorial`｛関数｝｛かんすう｝が
あります。

    factorial :: Int -> Int
    factorial 0       = 1
    factorial (n + 1) = (n + 1) * factorial n

｛次｝｛つぎ｝のように｛変換｝｛へんかん｝できます。

    factorial :: Int -> Int
    factorial 0 = 1
    factorial n = n * factorial (n - 1)
